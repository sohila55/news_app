## 📱 About The App

A modern news mobile application built with Flutter that fetches real live news from a REST API. The app delivers a smooth and clean reading experience with search functionality and a full news details view — all powered by Cubit state management and real API integration.

---

## ✨ Features

   ** Done :
- 🏠 **Home** — Browse latest real-time news fetched from REST API
- 📄 **News Details** — Full article view with image, title and content
- 🔍 **Search** — Search for any news topic in real time
  
   ** Will Added soon :
- 👤 **Profile** — View and manage user information

## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed
- News API key from [newsapi.org](https://newsapi.org)

### Installation

1. Clone the repo
```bash
git clone https://github.com/sohila55/news_app.git
```

2. Install dependencies
```bash
flutter pub get
```

3. Add API key
- Open `lib/core/constants/api_constants.dart`
- Replace `YOUR_API_KEY` with your actual API key

4. Run the app
```bash
flutter run
```
